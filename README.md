# 智慧银行实验课程

四川农业大学经济学院·数字经济系 | 2026 春季学期

## 项目简介

本项目为《中小商业银行经营管理》课程的实验教学资料仓库，包含 AI 编程工具实验讲义、环境配置指南、金融分析报告等教学资源。
