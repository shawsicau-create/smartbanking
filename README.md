# 中小商业银行课程

智慧银行实验课程资料

## 项目简介

本项目包含中小商业银行课程的实验讲义、AI开发环境配置指南和相关教学资源，托管于 [CNB 平台](https://cnb.cool/xiaosicau/smartbanking)。

## 快速开始

1. 克隆仓库：`git clone https://cnb.cool/xiaosicau/smartbanking.git`
2. 阅读 `智慧银行实验讲义_v3.0.pdf` 了解课程内容
3. 按照 `AI开发环境配置讲义_中文版.pdf` 配置开发环境
4. 参考 `实验手册.docx` 完成实验操作
