🎯 课程简介

本教程以 AI 编程工具（Qoder 等）为核心实验平台，系统讲授 MCP + Skill + CLI 技术栈在银行业务场景中的应用，通过 BMAD 方法论驱动从需求到部署的全流程项目实践。
本书定位：金融学专业本科生的 AI 工具实践教材。采用"低代码、高集成"的技术路线 —— 学生不需要从零编写 AI 程序，而是通过配置 IDE、接入 MCP 协议、编写 Skill 的方式，像搭积木一样组装出面向金融业务的智能系统。

🛠️ 技术栈

教学网站：Astro 6 + Starlight + Pagefind
AI 教学工具：Qoder / Trae CN / Cursor
工具协议：MCP (Model Context Protocol)
部署平台：EdgeOne Pages (腾讯云全球 CDN)
CI/CD：CNB 云原生流水线 (.cnb.yml)

🚀 本页面的部署方式

本页面是使用 edgeone-makers-mcp-server 的 deploy-html 工具部署的演示页面。

📦 部署命令（伪代码）：
CallMcpTool( server="edgeone-makers-mcp-server", tool="deploy-html", args={ value: "<html content>" } )
🌐 完整部署（多页面）使用 CLI：
cd webversion/ pnpm install && pnpm build source ../.env npx edgeone makers deploy ./dist -n smartbanking \ -t $EDGEONE_PAGES_API_TOKEN
