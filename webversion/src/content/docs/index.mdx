## 教学特色

:::tip[双线并行]
全书贯穿两条主线——**AI工具线**（IDE操作、MCP协议、Skill编写、BMAD方法论）和**金融业务线**（零售银行、公司银行、风控合规、数据分析），每一章都是"工具+场景"的有机融合。
:::

:::note[三层递进]
全书分为**基础模块**（第1—5章）、**进阶模块**（第6章）和**综合模块**（第7—8章），形成"认知→应用→创造"的能力递进路径。
:::

:::tip[理论实操并重]
每一章均包含理论讲解和实操练习两部分。理论部分提供必要的知识框架，实操部分给出分步操作指引，确保"学了就能用、用了就有效"。
:::
