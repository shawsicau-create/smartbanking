### 学术履历

庹国柱教授是中国农业保险研究领域的开拓者和奠基人之一，从事农业保险教学和研究近40年。1984年受中国人保陕西分公司邀请开始农业保险研究，1989年起开始招收保险方向研究生，1996年调至北京财贸学院（现首都经济贸易大学），深度参与中国农业保险改革发展的重大进程。

## 核心著作

### 一、学术专著

1. **《中国农业保险与农村社会保障制度研究》**
   - 作者：庹国柱、王国军
   - 出版社：首都经济贸易大学出版社
   - 出版年份：2002年12月
   - 获奖情况：2004年获北京市第八届哲学社会科学优秀成果一等奖
   - 核心贡献：系统论证了农业保险的准公共物品属性，构建了政策性农业保险制度的理论框架，提出农业保险立法必要性和可行性的论证

2. **《农业保险》**
   - 出版社：中国人民大学出版社
   - 出版年份：2005年
   - 内容特点：全面阐述农业保险的理论基础、国际经验和实践问题

3. **《庹国柱农业保险文集》（四卷本）**
   - 总字数：180万字
   - 出版情况：持续更新，第四卷2025年出版
   - 内容涵盖：收录庹国柱教授40年农业保险研究成果，是研究中国农业保险发展历程的珍贵文献

4. **《中国农业保险发展报告》（系列年度报告）**
   - 主编：庹国柱
   - 出版单位：中国农业出版社
   - 时间跨度：2011-2024年（持续出版）
   - 内容特点：系统记录和分析中国农业保险年度发展状况、政策变化和市场动态

5. **《国外农业保险：实践、研究和法规》**
   - 出版年份：1996年
   - 内容特点：系统介绍国际农业保险发展经验，为中国农业保险制度建设提供国际视野

### 二、代表性论文

1. **《从40年政策变化喜看我国农业保险蓬勃发展》**
   - 发表刊物：《保险研究》2019年第2期
   - 核心观点：系统回顾1978-2018年中国农业保险政策演变，总结农业保险从恢复试办到蓬勃发展的历史经验

2. **《我国农业保险大灾风险分散制度展望》**
   - 发表时间：2014年
   - 核心观点：提出建立多层次农业保险大灾风险分散体系的框架设计，为后续中农再（中国农业再保险）的成立提供理论支撑

3. **《试论"中农再"建立的意义》**
   - 发表时间：2020年
   - 核心观点：论证中国农业再保险公司的性质定位（"负有政策功能的商业性公司"）和四大历史使命

4. **《关于中国政策性农业保险制度模式改革的思考》**
   - 合作者：冯文丽
   - 发表刊物：《保险研究》2024年第12期
   - 核心观点：分析现行"市场运作"制度模式下中央与地方政府决策视角差异，提出制度模式改革方向

5. **《值得探讨的几个农险产品创新问题》**
   - 发表刊物：《中国保险》2021年第1期
   - 核心观点：探讨天气指数保险、收入保险等创新型农业保险产品的设计原理和实践问题

6. **《政策性农业保险需要专门管理机构》**
   - 合作者：朱俊生
   - 发表时间：2007年
   - 核心观点：提出建立政策性农业保险专门管理机构的必要性和职能设计

7. **《农作物保险风险分区和费率分区问题的探讨》**
   - 合作者：丁少群
   - 发表刊物：《中国农村经济》1994年第8期
   - 核心贡献：国内较早提出并论证农业保险实行风险分区、差异化费率的必要性和可行性

## 理论贡献与核心观点

### 一、农业保险的准公共物品理论

庹国柱教授系统论证了农业保险的准公共物品属性，这是其最重要的理论贡献之一：

**核心论点**：
- 农业保险既非纯粹私人物品，也非纯粹公共物品，而是介于两者之间的"准公共物品"
- 农业保险具有"收益外溢"特征：社会效益高而自身效益低
- 由于准公共物品属性，纯粹市场化运作必然导致"市场失灵"

**政策含义**：
- 政府必须介入农业保险市场，通过财政补贴、税收优惠等方式支持农业保险发展
- 2007年财政部农业保险保费补贴制度的建立，正是基于这一理论认知

### 二、政策性农业保险制度框架理论

庹国柱教授构建了中国政策性农业保险制度的完整理论框架：

**制度边界理论**：
- 明确区分政策性农业保险与商业性农业保险
- 政策性农业保险是"政策项目或政策工具"，而非单纯的商业行为
- 强调"划清政府和市场的界限"，反对将保险当作"第二财政"

**制度模式选择**：
- 论证了"政府引导、市场运作"的公私合作（PPP）模式的必要性和可行性
- 主张"市场运作、政府扶持"的制度安排
- 强调地方政府"不参与农业保险的具体经营"

### 三、农业保险"供需双冷"理论

庹国柱教授深刻揭示了中国农业保险市场的"供需双冷"困境：

**需求侧**：
- 低农业发展水平与农民收入水平同高昂保险费用之间存在矛盾
- 农户投保意愿低，存在"政府需要保险而农户投保热情低"的矛盾

**供给侧**：
- 保险公司因高成本低收益不愿供给
- 农业风险的特殊性（系统性风险、信息不对称、道德风险等）导致商业保险供给不足

**解决方案**：
- 2007年后实行保费补贴及税收优惠，推动市场快速增长
- 2023年保费补贴达966亿元，中国农业保险市场规模跃居世界第一

### 四、农业保险风险分区与差异化费率理论

1994年，庹国柱教授与丁少群合作，在国内较早提出并论证了农业保险实行风险分区、差异化费率的必要性和可行性：

**核心观点**：
- 农业生产风险具有明显的地域差异性
- 统一费率既不公平也不效率
- 应根据风险区划确定差异化费率，实现"风险与费率相匹配"

**实践意义**：
- 为农业保险精算定价提供理论基础
- 推动农业保险从"粗放定价"向"精准定价"转变

### 五、多层次农业保险大灾风险分散体系理论

庹国柱教授系统设计了农业保险大灾风险分散的多层次体系：

**三层风险分散机制**：
1. **第一层**：保险公司大灾风险准备金
2. **第二层**：农业再保险（中农再）
3. **第三层**：国家农业保险大灾风险准备金

**中农再的性质定位**：
- "负有政策功能的商业性公司"
- "采用市场化运作方式"，但"采用某些行政手段也是可以理解和可行的"

**四大历史使命**：
1. 为直保公司提供再保险服务，分散大灾风险
2. 推动建立并统筹管理国家农业保险大灾风险准备金
3. 建立农业保险信息共享平台
4. 推动农业补贴制度改革

### 六、农业保险政策目标体系理论

庹国柱教授与张峭合作，系统阐述了中国农业保险的政策目标：

**五大政策目标**：
1. 保障农业可持续发展与粮食安全
2. 促进农业现代化、保障农户收入稳定增长
3. 控制农产品质量、保障国民廉价粮食供给
4. 降低农产品价格、增强国际竞争力
5. 在脱贫攻坚和乡村振兴中发挥重要作用

## 心智模型（分析问题的独特视角）

### 一、"制度经济学"分析视角

庹国柱教授善于运用新制度经济学的理论工具分析农业保险问题：

**交易成本分析**：
- 从交易成本视角解释农业保险市场失灵的原因
- 分析政府干预如何降低交易成本、提高市场效率

**制度变迁分析**：
- 运用"路径依赖"理论解释中国农业保险制度演变的轨迹
- 分析制度变迁中的"锁定效应"和"路径突破"

### 二、"准公共物品"界定视角

庹国柱教授的核心分析框架是将农业保险界定为"准公共物品"：

**分析逻辑**：
- 物品属性 → 市场结果 → 政府角色 → 制度设计
- 通过界定农业保险的准公共物品属性，推导出政府干预的必要性和方式

### 三、"政府与市场边界"视角

庹国柱教授始终关注政府与市场的合理边界：

**核心原则**：
- "划清政府和市场的界限"
- 反对政府"越位"（直接干预经营）和"缺位"（监管不力）
- 主张"政府引导、市场运作"的公私合作模式

**实践批判**：
- 2023年就东北某县财政局违规干预理赔事件撰文批评
- 推动监管机构叫停不当行政干预

### 四、"国际比较-本土适配"视角

庹国柱教授注重国际经验与中国实际的结合：

**研究方法**：
- 系统研究美国、加拿大、日本等国农业保险制度
- 分析国际经验在中国的适用性和适配性
- 提出"借鉴但不照搬"的原则

### 五、"历史演进-制度逻辑"视角

庹国柱教授善于从历史演进中把握制度逻辑：

**研究特色**：
- 系统梳理中国农业保险40年发展历程
- 从政策演变中提炼制度变迁的规律
- 预测未来发展趋势和政策走向
