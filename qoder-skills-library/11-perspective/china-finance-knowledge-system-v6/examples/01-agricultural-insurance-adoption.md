# 案例01：农户参保决策分析

> **使用模型**：INS-A（农业保险参与决策模型）
> **学者视角**：Ashok Mishra（农场家庭模型）、David Just（行为决策）、庹国柱（制度供给）
> **问题类型**：D（农业保险）
> **检验命题**：INS1（准公共产品）、INS4（补贴道德风险）

---

## 用户问题

> "某省农业保险参保率长期徘徊在45%左右，远低于全国平均70%。请分析影响农户参保决策的关键因素，并提出提高参保率的政策建议。"

## 分析过程

### 第一步：模型选择

问题核心 = 农户参保决策 → 路由到 **INS-A**
辅助模型 = RF-A（农村金融供需匹配，用于制度层面补充）

### 第二步：调用学者视角

**Ashok Mishra的农场家庭模型视角：**
农户不是单纯的"理性经济人"，参保决策嵌入在家庭资源配置中。关键变量包括：
- 农场收入占比（收入越依赖农业，参保动机越强）
- 非农收入 diversification（非农收入高的农户自保能力更强，参保意愿可能反而降低）
- 风险态度测量（Pratt-Arrow系数）

**David Just的行为决策视角：**
农户对概率的认知存在系统性偏差：
- 灾害可得性偏差：近年来未受灾的农户会低估风险概率
- 损失厌恶：对保费支出的敏感度高于对潜在赔付的估值
- 现状偏好：习惯于不参保状态，改变需要外部推力

**庹国柱的制度供给视角：**
参保率低的根源不仅在于需求侧，更在于制度供给：
- "供需双冷"：供给端产品设计不匹配，需求端支付意愿不足
- 准公共产品属性：完全市场化必然供给不足（INS1命题）
- 信任机制：历史赔付纠纷导致的信任赤字

### 第三步：命题检验

| 命题 | 结论一致性 | 说明 |
|------|-----------|------|
| INS1 准公共产品属性 | 一致 | 45%参保率说明市场化供给不足，需要政府补贴 |
| INS4 补贴道德风险 | 需关注 | 提高补贴率时需配套风险管理要求 |
| INS8 数字技术影响 | 机会 | 遥感定损可降低信任赤字 |

### 第四步：结构化输出

```
## 农户参保决策分析：某省参保率45%的低 participation 诊断

### 分析框架
- 使用模型：INS-A（农业保险参与决策模型）
- 学者视角：Mishra（家庭资源配置）+ Just（行为偏差）+ 庹国柱（制度供给）
- 检验命题：INS1、INS4、INS8

### 核心分析

**因素1：保费负担与补贴率不足**
理论依据：INS-A模型中"保费补贴"是核心变量
数据推断：若全国平均补贴率65%对应70%参保率，45%参保率可能对应补贴率不足50%
命题检验：INS1支持——农业保险的准公共产品属性意味着市场均衡参保率必然偏低

**因素2：行为偏差放大效应**
理论依据：Just的选择架构理论
机制：可得性偏差使农户低估灾害概率，损失厌恶放大保费痛感
推论：单纯提高补贴率未必能线性提升参保率

**因素3：信任赤字**
理论依据：庹国柱"供需双冷"框架
机制：历史定损纠纷 → 信任度下降 → 参保意愿降低
推论：信任因素在INS-A模型中权重可能高于预期收益

### 政策启示

1. **补贴结构调整**（可行性：高）
   - 提高省级财政补贴比例，但将补贴与农户风险管理投入挂钩（防范INS4道德风险）

2. **助推设计**（可行性：中）
   - 村级参保示范户制度（利用社会学习机制）
   - 简化条款信息呈现（降低信息可得性成本）

3. **信任修复**（可行性：中，周期长）
   - 引入第三方定损（遥感+村委会双确认）
   - 建立快速理赔通道并公示赔付率

### 局限性
- 本分析未使用该省的具体计量数据，结论基于理论推演
- INS-A模型假设农户有基本的风险认知，极度贫困地区可能不适用
- 信任因素的量化需要实地调查支撑
```
