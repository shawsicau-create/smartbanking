# 中国金融学自主知识体系 · 分析副驾驶

> 从"被查的词典"到"被用的副驾驶"——装上它，你的Agent就戴上了马克思主义政治经济学+25位学者心智模型的眼镜。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 你什么时候需要它？

- 你要分析中国金融政策/改革，但不想套用西方金融学标准答案
- 你要诊断农业保险市场失灵，需要系统性风险理论+制度经济学双视角
- 你要评估农户参保决策，需要期望效用理论+行为经济学+制度供给三重框架
- 你要写保险学/农村金融方向的论文，需要从25位高引用学者的视角切入

## 它会交付什么？

装上后，当你问它任何金融/保险/农村金融问题，它会输出一份**结构化分析框架**：

```
## [问题标题]
### 分析框架 → 使用模型 + 学者视角 + 检验命题
### 核心分析 → 3-5个关键论点，每个标注理论依据
### 命题检验 → 结论是否与核心命题一致
### 政策启示 → 可操作建议，标注可行性等级
### 局限性 → 本分析的假设和边界
```

**具体示例**：见 [examples/01-agricultural-insurance-adoption.md](examples/01-agricultural-insurance-adoption.md)

## 快速开始

```bash
npx skills add your-org/china-finance-knowledge-system-v6
```

装完直接说：
- "用中国金融学视角分析利率市场化改革"
- "诊断某地区农业保险市场失灵的根源"
- "用INS-A模型分析农户参保决策影响因素"

## 它和同类有什么不同？

| 对比维度 | 本Skill | 投研类Skill（如A股分析） | 通用金融Skill |
|---------|---------|----------------------|-------------|
| 定位 | **理论分析副驾驶** | 投研工具 | 数据查询 |
| 知识资产 | 22模型+23命题+25学者+128文献 | 估值模型/财报分析 | 基础概念 |
| 保险学覆盖 | 独家——5个专用模型+10位国际学者 | 无 | 无或极浅 |
| 理论根基 | 马克思主义+中国化+古代智慧 | 西方金融学 | 西方金融学 |
| 输出 | 结构化分析框架 | 研报/估值 | 数据/指标 |

## 安全边界

**不会做**：保单精算定价、个股分析、金融市场预测、替代学术论文
**会暂停**：问题无法分类时、需要数据但来源不确定时、结论与预期冲突时

详见 [SKILL.md](SKILL.md) 的 Checkpoints 和 Safety Boundaries 节。
