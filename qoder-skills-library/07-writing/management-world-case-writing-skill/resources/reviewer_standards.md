# 《管理世界》审稿标准

## 一、选题标准

高水平选题必须满足：

1. 面向中国重大现实问题；
2. 不是一般经验介绍；
3. 有理论反常或解释缺口；
4. 能与国际或国内重要理论对话；
5. 有理论建构空间。

## 二、理论标准

审稿人重点判断：

1. 是否明确对话某一理论脉络；
2. 是否指出既有研究真正不足；
3. 是否提出新的概念、机制、模型或边界条件；
4. 是否只是用中国案例重复已有理论。

## 三、方法标准

必须关注：

1. 案例选择逻辑是否充分；
2. 一手资料是否扎实；
3. 访谈和文本资料是否充分；
4. 是否说明编码过程；
5. 是否进行三角验证；
6. 数据是否支撑理论结论。

## 四、发现标准

好的发现应当是：

1. 有阶段；
2. 有机制；
3. 有概念命名；
4. 有模型；
5. 有张力；
6. 有边界条件。

差的发现通常是：

1. 工作经验总结；
2. 政策建议罗列；
3. 访谈材料堆砌；
4. 理论概念空转；
5. 案例事实与理论结论脱节。

## 五、写作标准

《管理世界》偏好：

1. 问题意识强；
2. 逻辑链条清晰；
3. 概念表达准确；
4. 语言克制；
5. 理论贡献明确；
6. 篇幅与贡献匹配。

## 六、审稿结论类型

审稿时可以给出：

1. 建议录用；
2. 修改后录用；
3. 大修后再审；
4. 不建议录用。

一般情况下，对博士生论文初稿应采用“大修后再审”的标准进行严格评估。
