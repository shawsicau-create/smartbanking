## Stata Variable Coding Reference

```stata
* === Core Variables ===
gen gender = 1 if A6 == "Male" | A6 == "男"
replace gender = 0 if A6 == "Female" | A6 == "女"
recode A8 (0=0) (1=1) (2=2) (3=3) (4=4), gen(education)

* === Financial Inclusion Index ===
egen digital_fi = rowtotal(C1a C1b C1d C1e C1l), missing
gen has_credit = (C1g > 0)
gen formal_credit = inlist(C1f_source, 1, 3, 4)
gen informal_credit = inlist(C1f_source, 2, 5)

* === Training Exposure ===
gen training = (C2a == 1)
gen training_freq = C2b
gen coop_member = (C2f == 1)
gen joint_effect = digital_fi * training  // Key interaction term

* === Technology Adoption Index ===
egen tech_adopt = rowtotal(D1a D1b D1c D1d D1e D1f D1g D1h D1i D1j), missing

* === Social Capital Index ===
egen social_capital = rowtotal(D2a D2b D2c D2d D2e D2f D2g D2h), missing
gen trust_fi = D2j  // Trust in financial institutions
gen trust_market = D2l  // Trust in buyers/millers

* === Food Security Score ===
egen fies_score = rowtotal(E2a E2b E2c E2d E2e E2f E2g), missing
gen food_insecure = (fies_score >= 2)  // Binary: moderately/severely food insecure
gen fcs_score = E2h  // Food Consumption Score
gen hdds = E2i  // Dietary Diversity Score

* === Log transformations for skewed variables ===
gen ln_rice_income = ln(B3d + 1)
gen ln_net_income = ln(B3h + 1)
gen ln_loan = ln(C1g + 1) if C1g > 0
gen ln_yield_acre = ln(E1a + 1)

* === Control Variables ===
gen age = A7
gen age2 = age^2
gen hhsize = A11
gen rice_area_acre = B1c
gen log_asset = ln(H_total_asset_value + 1)

* === Moderating Variables ===
gen subsidy = (F1a == 1)
gen market_access = (F2e == 1)
gen market_distance = F1g

* === ABM Parameters ===
gen risk_averse = (G1a == 2 & G1b == 2)  // Risk-averse if chooses B in both
gen risk_seeking = (G1a == 2 & G1b == 1)  // Risk-seeking if chooses B only in gain
gen time_pref_rate = ((120/100)^(12/1) - 1) if G2a == 2  // Monthly discount factor
gen herding = (G3c >= 4)  // High herding behavior if >=4
gen trust_general = D2j

* === Panel Setup (if 3-year recall data) ===
encode HHID, gen(hh_id)
encode year, gen(year_id)
xtset hh_id year_id

* === Summary Statistics ===
summarize digital_fi training joint_effect tech_adopt social_capital fies_score ln_net_income, d

* === Correlation Matrix ===
pwcorr digital_fi training joint_effect tech_adopt social_capital fies_score ln_net_income, star(0.05)

* === Fixed Effects Regression ===
xtreg ln_net_income c.digital_fi##c.training tech_adopt social_capital i.year, fe robust
estimates store fe_main

* === Mediation Analysis (3-step) ===
* Step 1: Total effect
xtreg fies_score c.digital_fi##c.training i.year, fe robust

* Step 2: Mediators on IV
xtreg tech_adopt c.digital_fi##c.training i.year, fe robust
xtreg social_capital c.digital_fi##c.training i.year, fe robust

* Step 3: Joint model
xtreg fies_score c.digital_fi##c.training c.tech_adopt c.social_capital i.year, fe robust

* === Robustness Checks ===
winsor2 ln_net_income fies_score, cuts(1 99) replace
xtreg ln_net_income_w c.digital_fi##c.training tech_adopt social_capital i.year, fe robust

* === Heterogeneity Analysis ===
xtreg ln_net_income c.digital_fi##c.training if gender == 0, fe robust  // Female-headed
xtreg ln_net_income c.digital_fi##c.training if subsidy == 1, fe robust  // With subsidy
```

## Sampling Note (Reference: Cochran 1977)

- **Target population**: Smallholder rice farmers in Northern/North East Ghana
- **Estimated N**: ~15,000 rice-farming households
- **Formula**: $n_0 = Z^2 \cdot p(1-p) / e^2 = 1.96^2 \times 0.5 \times 0.5 / 0.05^2 = 384$
- **Design effect (Deff=1.5 for cluster sampling)**: $n_{adj} = 384 \times 1.5 = 576$
- **With 10% non-response**: $n_{final} = 576 / 0.9 = 640$ households
- **Recommended sampling**: 2-stage cluster: 32 districts × 20 households each
