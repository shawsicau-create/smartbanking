## Stata Coding Example

```stata
* Demographics
recode A5 (1=0) (2=1) (3=2) (4=3) (5=4) (6=5), gen(education)
gen log_income = ln(总收入 + 1)

* Financial services access
gen has_account = (D1 == 1)
gen has_credit = inlist(E6, 4, 5)
gen has_insurance = (F1 == 1)

* Satisfaction indices (average multiple items)
alpha D4 F2 G2 H3, gen(satisfaction_index)
```

## Quality Assurance

1. **Pre-test**: 20-30 respondents, timing, comprehension check
2. **Skip logic verification**: Every path tested
3. **Data consistency checks**: 
   - Income = Sum of components?
   - Assets - Liabilities = Net worth?
4. **Panel tracking**: Unique household ID + year code
