## Stata变量编码说明

```stata
* === 数据编码准备 ===
destring age hhsize labor tea_area, replace
destring income_* cost_* revenue_*, replace

* 核心被解释变量
gen net_income = revenue_total - cost_total  // 净收益
gen tea_income_ratio = tea_income / total_income  // 茶叶收入占比
gen log_net_income = ln(net_income + 1)  // 对数净收益

* 核心解释变量：金融可得性
gen has_credit = (loan_amount > 0)  // 是否获得贷款
gen credit_amount = loan_amount  // 贷款金额
gen credit_access = (credit_limit > 0)  // 是否有授信额度
gen informal_credit = (informal_loan > 0)  // 非正规借贷

* 中介变量
gen tech_adopt = tech_irrig + tech_machine + tech_biopesticide  // 技术采用指数
gen social_network = num_tea_friends + wechat_groups  // 社会网络规模
gen risk_coping = has_insurance + has_savings + coop_member  // 风险应对能力

* 调节变量
gen policy_subsidy = subsidy_amount > 0  // 是否有政策补贴
gen market_perception = price_satisfaction + info_transparency  // 市场感知

* ABM行为参数
gen risk_aversion = (choice_a == 1)  // 风险规避者=1
gen time_preference = months_wait / 12  // 时间偏好率
gen herding_behavior = (follow_neighbors == 1)  // 从众行为

* 控制变量
gen age2 = age^2  // 年龄平方项（可能非线性）
gen edu_level = recode(education, 1,2,3,4,5)  // 教育水平

* 描述性统计
summarize net_income has_credit tech_adopt social_network risk_aversion

* 计量模型
xtreg net_income has_credit tech_adopt social_network i.year, fe robust
eststo: xtreg log_net_income c.has_credit##c.tech_adopt, fe robust
esttab using "tea_farmer_results.csv", replace se star
```
