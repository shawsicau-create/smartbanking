## 一、抽样公式（关键文献）

### 1.1 Cochran 无限总体公式

用于总体规模未知或非常大的情况：

$$n_0 = \frac{Z^2 \cdot p \cdot (1-p)}{e^2}$$

**参数说明：**

| 符号 | 含义 | 常用取值 |
|------|------|----------|
| $n_0$ | 最小样本量 | 待计算 |
| $Z$ | 置信水平对应的Z值 | 95%→1.96；99%→2.58 |
| $p$ | 总体比例（特征比例） | 无先验时取0.5（最保守） |
| $e$ | 允许误差（边际误差） | 通常取0.05 |

**示例（雅安茶农）：**

已知：置信水平95%（Z=1.96），无先验信息（p=0.5），允许误差e=0.05

$$n_0 = \frac{1.96^2 \times 0.5 \times (1-0.5)}{0.05^2} = \frac{3.8416 \times 0.25}{0.0025} = \frac{0.9604}{0.0025} = 384.16 \approx 385$$

**结论：最小样本量 = 385 户**

---

### 1.2 有限总体校正公式 (FPC)

当总体规模N已知且较小时，需要用FPC校正：

$$n = \frac{n_0}{1 + \frac{n_0 - 1}{N}}$$

**参数说明：**

| 符号 | 含义 |
|------|------|
| $n_0$ | Cochran公式计算的初始样本量 |
| $N$ | 目标总体规模 |
| $n$ | 校正后的样本量 |

**示例（雅安茶农）：**

假设雅安茶农总体N = 5,000户，$n_0$ = 385

$$n = \frac{385}{1 + \frac{385-1}{5000}} = \frac{385}{1 + 0.0768} = \frac{385}{1.0768} = 357.5 \approx 358$$

**校正后样本量 = 358 户**

---

### 1.3 分层抽样样本分配

若按茶园规模或地区分层，各层样本量按比例分配：

$$n_h = n \times \frac{N_h}{N}$$

其中 $N_h$ 为第h层总体规模。

---

### 1.4 常见抽样设计参考

| 调查类型 | 推荐公式 | 适用场景 |
|----------|----------|----------|
| 比例估计 | Cochran无限总体 | 二分类变量（有无信贷、保险等） |
| 均值估计 | $n = \frac{Z^2 \cdot \sigma^2}{e^2}$ | 连续变量（收入、成本等） |
| 有限总体 | Cochran + FPC | 总体规模已知 |
| 分层抽样 | 按比例分配 | 子群体分析 |
| 整群抽样 | $n_{cluster} = n \times \bar{m}$ | 村为群单位 |

## 二、样本量计算（雅安茶农案例）

### 2.1 关键参数设定

| 参数 | 设定值 | 依据 |
|------|--------|------|
| 置信水平 | 95% (Z=1.96) | 学术标准 |
| 允许误差 | 5% (e=0.05) | 可接受精度 |
| 总体比例p | 0.5（无先验，最保守） | Cochran 1977建议 |
| 目标总体N | 雅安市茶农约5,000户 | 雅安农业局统计 |
| 设计效应Deff | 1.5（考虑整群抽样） | 经验值 |

### 2.2 样本量计算

**Step 1：基础样本量**
$$n_0 = \frac{1.96^2 \times 0.5 \times 0.5}{0.05^2} = 385$$

**Step 2：设计效应调整**
$$n_{adj} = n_0 \times Deff = 385 \times 1.5 = 577.5 \approx 578$$

**Step 3：有限总体校正（FPC）**
$$n_{final} = \frac{578}{1 + \frac{578-1}{5000}} = \frac{578}{1.1154} = 518.3 \approx 519$$

**最终样本量：519 户**

**容许10%-15%无回答率**：519 × 1.15 = **597 ≈ 600户**

### 2.3 分层抽样设计

| 分层维度 | 分层标准 | 预计样本分配 |
|----------|----------|-------------|
| 地区 | 名山区、雨城区、荥经县等 | 按茶园面积比例分配 |
| 茶园规模 | <10亩 / 10-30亩 / >30亩 | 按1:2:1比例 |
| 收入水平 | 低/中/高 | 按 tercile 分组 |

### 3.3 抽样方案

#### 抽样框架

```
雅安市茶农总体 (N≈5,000)
    ↓ 分层（地区 × 茶园规模）
    ↓ PPS抽样（各村茶农数量加权）
    ↓ 整群抽样（村为群，每群抽取15-20户）
```

#### 抽样步骤

1. **获取村级名录**：从雅安农业农村局获取各村茶农名单
2. **分层**：按地区×茶园规模交叉分层
3. **确定村级样本量**：PPS法（按茶农数量比例分配）
4. **村级内抽样**：随机抽取15-20户
5. **替代户**：每村多抽5户作为替补

#### 抽样公式（参考）

| 步骤 | 公式 | 说明 |
|------|------|------|
| 村级分配 | $n_h = n \times \frac{N_h}{N}$ | 按茶农数量比例 |
| 村级内抽取 | 系统抽样 | 按户籍名册等间隔抽取 |

#### 双录入校验流程

```
录入员A录入数据 ──┐
                   ├──→ 比较差异 ──→ 核对原始问卷 ──→ 更正
录入员B录入数据 ──┘
```

#### EpiData 操作步骤

1. 创建 .QES 文件（定义变量和格式）
2. 生成 .REC 文件（数据文件）
3. 双人独立录入
4. 使用 CHECK 模块的 REVISION 报告比较差异
5. 根据原始问卷更正不一致处
6. 输出 CLEAN.DTA（Stata格式）

#### 数据清洗代码（Stata）

```stata
* 检查异常值
sum net_income, d
list if net_income < 0  // 净收益不能为负
list if net_income > 1000000  // 极端高值核查

* 检验一致性
assert credit_used == 1 if loan_amount > 0  // 有贷款必有使用
assert loan_amount < credit_limit if loan_amount > 0  // 贷款≤额度

* 缺失值报告
misstable summarize
```

### 3.5 计量分析

#### 分析框架

```
描述性统计
    ↓
相关性分析
    ↓
面板固定效应回归
    ↓
中介效应检验
    ↓
稳健性检验
```

#### Stata分析代码模板

```stata
*--------- 1. 描述性统计 ---------
use "yaan_tea_farmers.dta", clear
summarize net_income has_credit tech_adopt social_network ///
          risk_aversion age education land_size, d

*--------- 2. 相关性矩阵 ---------
pwcorr net_income has_credit tech_adopt social_network, star(0.05)

*--------- 3. 面板固定效应 ---------
xtset household_id year
xtreg net_income has_credit tech_adopt social_network i.year, fe robust
estimates store fe_main

*--------- 4. 中介效应检验 ---------
* 步骤1：总效应
xtreg net_income has_credit i.year, fe robust

* 步骤2：中介→自变量
xtreg tech_adopt has_credit i.year, fe robust
xtreg social_network has_credit i.year, fe robust

* 步骤3：联合模型
xtreg net_income has_credit c.tech_adopt c.social_network i.year, fe robust

*--------- 5. 调节效应 ---------
gen has_credit_X_policy = has_credit##c.policy_subsidy
xtreg net_income c.has_credit##c.policy_subsidy i.year, fe robust

*--------- 6. 稳健性检验 ---------
* 替换因变量
xtreg ln_tea_income has_credit tech_adopt i.year, fe robust

* 缩尾处理极端值
winsor2 net_income, cuts(1 99) replace
xtreg net_income_w has_credit tech_adopt i.year, fe robust

*--------- 7. 异质性分析 ---------
xtreg net_income has_credit tech_adopt if land_size > 10, fe robust
xtreg net_income has_credit tech_adopt if coop_member == 1, fe robust
```

#### 核心变量编码

| 变量名 | 定义 | 来源题号 |
|--------|------|----------|
| net_income | 净收益 = 总收入 - 总成本 | Q2-24 |
| has_credit | 是否获得贷款（0/1） | Q4-5~4-7 |
| credit_amount | 贷款金额（万元） | Q4-5~4-7 |
| tech_adopt | 技术采用指数（加总） | Q5-1~5-6 |
| social_network | 社会网络规模（户数） | Q5-9 |
| risk_aversion | 风险规避（0=中性，1=规避） | Q7-1, Q7-2 |
| time_pref | 时间偏好率 | Q7-3, Q7-4 |
| policy_subsidy | 政策补贴金额 | Q6-1~6-3 |
| land_size | 茶园面积（亩） | Q2-1 |

#### Stata参数提取代码

```stata
*--------- ABM参数提取 ---------
use "yaan_tea_farmers.dta", clear

* 1. 风险规避比例
gen risk_averse = (q7_1_choice == 1 & q7_2_choice == 1)
summarize risk_averse
local p_risk = r(mean)

* 2. 时间偏好分布
* 根据选择计算贴现因子
gen delta = cond(q7_4_choice==2, 0.9, 0.7)  // 1年后多50%≈δ=0.9
summarize delta

* 3. 土地规模分布
summarize land_size, d
local mu_land = r(mean)
local sd_land = r(sd)

* 4. 合作社参与率
tab coop, gen(coop_dummy)
local p_coop = r(mean)

* 5. 输出参数表
putexcel set "abm_parameters.xlsx", replace
putexcel A1 = "Parameter" B1 = "Value" C1 = "Distribution"
putexcel A2 = "risk_aversion_rate" B2 = `p_risk' C2 = "Bernoulli"
putexcel A3 = "mean_land_size" B3 = `mu_land' C3 = "Gamma"
putexcel A4 = "coop_participation" B4 = `p_coop' C4 = "Bernoulli"
```
