# 更新日志

本文件记录 PaperFinder 文献综述助手技能的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-02-03

### 新增
- ✨ 初始版本发布
- 🔍 集成 Allen AI PaperFinder 进行智能文献检索
- 📊 基于多维度指标的文献质量评估系统
- 📝 自动化文献综述报告生成
- 📥 从 arXiv 等开放获取源自动下载 PDF
- 📚 BibTeX 格式引用信息自动生成
- 🗂️ 文献分类组织（基础、最新、方法、应用、综述）
- 📋 支持 Markdown 和 JSON 两种输出格式
- ⚙️ 可配置参数（论文数量、年份范围、下载选项）
- 🔒 安全的文件系统和网络访问控制
- 🚀 Agent 型自主工作流程
- 🌐 支持跨平台（macOS, Linux, Windows）

### 功能特性
- 智能语义搜索和文献匹配
- 引用数量和期刊质量评估
- 时间范围筛选
- 自动识别开放获取来源
- 结构化报告生成
- 研究趋势分析
- 研究空白识别

### 文档
- 📖 完整的 SKILL.md 规范文件
- 📘 详细的 README 使用指南
- 📝 丰富的使用示例文档（EXAMPLES.md）
- 📄 MIT 开源许可证
- 📋 版本更新日志

### 技术规格
- 最低 Claude Code 版本：1.0.0
- 默认超时时间：600 秒
- 缓存 TTL：3600 秒
- 默认 AI 模型：claude-sonnet-4-20250514
- Temperature：0.3
- Max Tokens：8000

## 版本说明

### 语义化版本格式
- **主版本号 (MAJOR)**: 不兼容的 API 变更
- **次版本号 (MINOR)**: 向后兼容的功能新增
- **修订号 (PATCH)**: 向后兼容的问题修复

### 变更类型标签
- `新增` - 新功能
- `变更` - 现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - Bug 修复
- `安全` - 安全性相关修复
