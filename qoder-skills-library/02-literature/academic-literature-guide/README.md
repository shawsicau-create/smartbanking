# 文献导读助手 (Academic Literature Guide)

专为大专/本科大一新生量身定制的学术文献导读专家技能。

## 📦 安装

本技能已创建至：
```
~/Library/Application Support/LobsterAI/SKILLs/academic-literature-guide/
```

在 LobsterAI 中使用时，当用户提到以下场景会自动触发：
- "帮我读这篇文献"
- "看不懂论文"
- "文献导读"
- "大一新生读论文"
- "论文解读"
- "学术文献理解"

## 🎯 核心功能

### 1. 四层渐进理解
- **🟢 直觉层**：用生活场景建立连接
- **🔵 概念层**：术语白话解释 + 概念地图
- **🟡 技术层**：方法拆解（菜谱式步骤）
- **🔴 批判层**：培养科学怀疑精神

### 2. 强制事实检索
任何关于后续发展、批评、撤稿的陈述必须先完成 Web Search 并附权威链接。

### 3. 知识脉络可视化
必须输出知识族谱图，展示研究的前因后果。

### 4. 学习固化工具包
- 自测检查清单（基础/进阶/深度三级）
- 扩展阅读路径（分级推荐）
- 学习建议与工具推荐

## 🔧 辅助脚本

### search-citations.py
用于检索论文的后续引用、批评、复现、撤稿信息。

用法：
```bash
python scripts/search-citations.py "作者姓氏" "年份" "关键词 1" "关键词 2" ...
```

示例：
```bash
python scripts/search-citations.py "Zimbardo" "1973" "prison" "guards"
```

## 📚 术语库

`references/freshman-vocab.md` 包含按学科分类的术语白话解释：
- 🧠 心理学（去个性化、角色内化、认知失调等）
- 💻 计算机科学（Attention、Q/K/V、Encoder-Decoder 等）
- 🧬 生物学（CRISPR、gRNA、脱靶效应等）
- 📊 统计学（p 值、置信度、相关性≠因果性等）
- 🎓 学术方法论（同辈评审、可重复性、效应量等）

在 Layer 2（概念层）中直接复用这些解释，确保术语密度符合新生水平。

## ⚠️ 生存铁律

使用本技能时必须遵守：

1. **强制事实检索** - 未检索不得编造后续发展
2. **新生友好表达** - 所有术语首次出现必须白话解释
3. **禁止幻觉** - 不确定内容标注 [待核验]
4. **四层递进** - 严格按直觉→概念→技术→批判顺序
5. **知识脉络** - 必须输出知识族谱图
