# Academic Researcher

## Purpose

执行系统性学术研究，从选题识别到文献综合和假说发展。适用于学术研究初期阶段的全流程指导。

## When to Use

- 开始新的学术研究项目时使用
- 从选题到假说形成的研究流程需要指导时触发
- 触发词："学术研究"、"研究流程"、"选题"

## Instructions

### Step 1
确认研究领域和初步方向
### Step 2
选题扫描：识别研究空白和前沿问题
### Step 3
文献系统性搜索与筛选
### Step 4
文献综合：提取核心观点和争论
### Step 5
假说形成：基于文献空白提出可检验假说

## Example Prompts

- "帮我系统梳理农村金融领域的研究前沿"
- "从文献空白中形成可检验假说"
- "设计一个完整的学术研究计划"

## Requirements

### Software
- Python 3.10+ or appropriate runtime

## Best Practices

1. **使用系统综述而非随意搜索**
2. **假说需有理论机制支撑**
3. **标注文献质量和可信度等级**

## Common Pitfalls

- ❌ 文献搜索过于狭窄
- ❌ 假说缺乏理论机制支撑
- ❌ 忽视对立观点和批评
