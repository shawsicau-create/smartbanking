## 核心资产

### ABM 经济系统仿真 (`05-analysis/abm-analyzer`)

基于代理模型（Agent-Based Modeling）的经济系统仿真分析平台：模型构建、参数校准、蒙特卡洛模拟、学术级可视化。

### 中国金融学自主知识体系 (`11-perspective/china-finance-knowledge-system-v6`)

以马克思主义政治经济学为根基，融合三千年中国金融思想的活体知识系统。22 个分析模型 · 23 条核心命题 · 12 条决策启发式 · 8 大学术流派。

### 女娲 Skill 造人术 (`11-perspective/nuwa-skill`)

自动深度调研 → 思维框架提炼 → 生成可运行的人物思维 Skill。已蒸馏 15 位人物（Munger、Taleb、Feynman、Musk、Naval 等）。

## 如何使用

1. **自动匹配**：在 Qoder 中发出自然语言请求，AI 自动加载对应 Skill
2. **显式调用**：通过 `/skill-name` 直接触发
3. **安装路径**：`~/.qoder/skills/`（全局）或项目内 `.qoder/skills/`（项目级）
4. **思维切换**：说"用XX的视角"即可激活对应人物 Skill
