# Consulting Playbook

## Purpose

提供咨询项目方法论手册，从问题定义到建议交付的完整流程。适用于战略咨询项目、管理学案例分析和企业诊断项目。

## When to Use

- 需要按咨询方法论执行战略项目时使用
- 管理学案例研究需要结构化分析流程时触发
- 触发词："咨询方法论"、"项目流程"、"consulting playbook"

## Instructions

### Step 1
确认项目类型：战略/运营/组织/数字化
### Step 2
Phase 1: 问题定义与范围界定
### Step 3
Phase 2: 数据收集与诊断分析
### Step 4
Phase 3: 方案设计与选项评估
### Step 5
Phase 4: 建议交付与实施路线图

## Example Prompts

- "按咨询方法论诊断零售企业战略问题"
- "设计数字化转型咨询项目流程"
- "组织诊断项目的分析框架"

## Requirements

### Software
- Python 3.10+ or appropriate runtime

## Best Practices

1. **80/20原则：聚焦关键驱动因素**
2. **假设驱动而非数据漫游**
3. **每阶段设置明确的交付物**

## Common Pitfalls

- ❌ 数据漫游无方向
- ❌ 忽视利益相关方视角
- ❌ 建议缺乏实施可行性
