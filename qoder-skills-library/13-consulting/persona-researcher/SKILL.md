# Persona Researcher

## Purpose

创建和分析用户画像(Persona)，通过行为细分进行市场研究和产品战略分析。适用于市场细分、用户研究和产品定位。

## When to Use

- 需要构建用户画像时使用
- 市场细分和用户研究时触发
- 触发词："用户画像"、"persona"、"市场细分"

## Instructions

### Step 1
确认研究目标：产品设计/营销定位/市场细分
### Step 2
收集行为数据：使用场景、决策因素、痛点
### Step 3
聚类分析识别行为模式
### Step 4
构建Persona档案(3-7个典型画像)
### Step 5
映射产品功能与Persona需求

## Example Prompts

- "为在线教育产品构建3个核心Persona"
- "分析SaaS用户的行为细分模式"
- "创建B2B买家决策旅程"

## Requirements

### Software
- Python 3.10+ or appropriate runtime

## Best Practices

1. **数据驱动而非凭直觉创建Persona**
2. **优先区分行为差异而非人口统计差异**
3. **每个Persona需有明确的痛点和使用场景**

## Common Pitfalls

- ❌ Persona过多失去焦点(3-7个为宜)
- ❌ 混淆人口统计和行为画像
- ❌ 忽视边缘用户但重要的群体
