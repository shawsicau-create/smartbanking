# Economic Systems

## Purpose

使用比较经济学和制度分析框架分析经济体系和制度结构。适用于制度经济学研究、经济体制比较和发展路径分析。

## When to Use

- 分析经济体制和制度结构时使用
- 制度经济学研究时触发
- 触发词："经济体系"、"制度分析"、"比较经济学"

## Instructions

### Step 1
确认分析框架：新制度经济学/比较经济学/政治经济学
### Step 2
识别关键制度变量：产权/市场结构/治理机制
### Step 3
比较不同经济体系的表现差异
### Step 4
分析制度变迁的驱动因素
### Step 5
生成制度分析报告与发展路径建议

## Example Prompts

- "比较中国和印度经济发展路径的制度差异"
- "分析产权制度对农村发展的影响"
- "评估市场化改革对资源配置效率的影响"

## Requirements

### Software
- Python 3.10+ or appropriate runtime

## Best Practices

1. **避免价值判断驱动的体系比较**
2. **注重历史语境和演进路径**
3. **区分正式制度和非正式制度**

## Common Pitfalls

- ❌ 将西方模式作为唯一标杆
- ❌ 忽视文化和社会因素
- ❌ 过度简化复杂的制度交互
