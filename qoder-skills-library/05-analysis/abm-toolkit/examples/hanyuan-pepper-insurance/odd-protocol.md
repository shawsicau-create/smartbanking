# 汉源花椒保险模拟实验 — ODD协议

> 基于 Grimm et al. (2006) ODD协议标准，聚焦汉源县花椒产业特色

---

## 1. Overview（概述）

### 1.1 研究目的

模拟汉源县花椒种植户在"倒春寒"、干旱、收获期暴雨等多重气候风险下的保险参保决策动态，评估不同保费补贴策略和指数保险设计方案对参保率、农户福利和保险公司可持续性的影响。

**核心问题**：
- 汉源花椒种植户"供需双冷"的根源是什么？
- 倒春寒指数保险的最优触发温度如何设定？
- 阶梯式补贴 vs 统一补贴，哪个更有效？

### 1.2 主体类型

| 主体 | 数量 | 核心属性 | 核心行为 |
|------|------|----------|----------|
| 椒农 (PepperFarmer) | 300 | 海拔、树龄、土地规模、风险态度、社会网络 | 投保决策、生产投入、社会学习 |
| 保险公司 (Insurer) | 1 | 资本金、准备金、赔付率 | 承保、理赔、费率调整 |
| 政府 (Government) | 1 | 财政预算、政策目标 | 保费补贴、大灾兜底 |

### 1.3 空间结构

- **混合模式**：20×20网格模拟汉源山区地形
- **海拔分层**：低山带（700-1200m）、中山带（1200-1800m）、高山带（1800-2500m）
- **社交网络**：Moore邻域（8邻居）模拟邻里效应，半径=2格

### 1.4 时间尺度

- 1个时间步 = 1个种植季（约1年）
- 仿真总步数：30步（30个种植季）
- 花椒生长周期：定植后3年挂果，5年丰产

## 2. Design Concepts（设计概念）

### 2.1 涌现

- 宏观参保率的S型扩散曲线
- 高海拔椒农率先参保的"梯度效应"
- 极端气候冲击下的参保率跳升与回落

### 2.2 适应

- 椒农基于上年收益和邻居行为更新投保策略（社会学习+强化学习）
- 损失经历更新风险感知（贝叶斯更新）

### 2.3 交互

- **横向**：邻里观察（邻居获赔→投保意愿+15%）
- **纵向**：保险-椒农理赔合约、政府-保险公司补贴合约

### 2.4 异质性

| 维度 | 分布 | 参数 |
|------|------|------|
| 海拔 | 均匀分布 | Uniform(700, 2500)m |
| 土地规模 | 指数分布 | Exponential(5)+1 亩 |
| 树龄 | 均匀分布 | Uniform(1, 20)年 |
| 风险厌恶 | 均匀分布 | CRRA ρ ~ Uniform(0.5, 2.0) |
| 初始资产 | 对数正态 | LogNormal(10.5, 0.5) |

### 2.5 随机性

- **倒春寒**：年发生概率 p=0.20，损失率 Uniform(0.2, 0.7)
- **干旱**：年发生概率 p=0.12，损失率 Uniform(0.1, 0.4)
- **收获期暴雨**：年发生概率 p=0.15，损失率 Uniform(0.15, 0.5)
- **花椒价格波动**：几何布朗运动 dP = μPdt + σPdW, μ=0.02, σ=0.15

## 3. Details（细节）

### 3.1 初始化

- 椒农数量: 300（对应汉源县典型花椒乡镇规模）
- 初始投保率: 0%（冷启动）
- 基准补贴率: 30%（参考四川省政策性农险标准）
- 基准保费率: 5%（花椒保险精算费率）

### 3.2 花椒产量模型

```
亩产 = base_yield(altitude, tree_age) × weather_shock × management_factor
其中：
  base_yield = f(海拔, 树龄)  # 海拔1200-1800m最优，树龄5-15年丰产
  weather_shock ~ 气候冲击因子
  management_factor ~ 管理水平 N(1.0, 0.1)
```

### 3.3 投保决策模型

```
P(insure) = 1 / (1 + exp(-(β₀ + β₁·subsidy + β₂·neighbor_rate + β₃·loss_exp + β₄·altitude_risk)))

默认参数：
  β₀ = -2.0    # 汉源花椒种植户基准倾向（低于通用值，反映经济作物认知差异）
  β₁ = 3.5     # 补贴敏感度（经济作物农户对补贴更敏感）
  β₂ = 2.0     # 邻里效应（山区信息闭塞，邻里影响更大）
  β₃ = 0.6     # 损失经历效应
  β₄ = 0.3     # 海拔风险感知
```

### 3.4 参数校准表

| 参数 | 符号 | 取值 | 来源 | 说明 |
|------|------|------|------|------|
| 倒春寒概率 | p_frost | 0.20 | 汉源县气象局统计 | 海拔>1200m显著增加 |
| 干旱概率 | p_drought | 0.12 | 四川省气候中心 | 春旱为主 |
| 暴雨概率 | p_rain | 0.15 | 收获期7-8月 | 影响晾晒品质 |
| 花椒基准价格 | P₀ | 80 元/斤 | 汉源花椒市场行情 | 贡椒品质 |
| 亩产基准 | Y_max | 150 斤/亩 | 丰产期均值 | 干椒产量 |
| 保费率 | r_premium | 0.05 | 政策性农险参考 | 经济作物 |
| 赔付比例 | indemnity_ratio | 0.75 | 标准条款 | 物化成本保险 |
| 损失厌恶系数 | λ | 2.25 | Kahneman & Tversky (1992) | 标准值 |
| 概率权重参数 | γ | 0.65 | Tversky & Kahneman (1992) | 高估小概率 |

### 3.5 情景设计

| 情景 | 描述 | 补贴率 | 气候 |
|------|------|--------|------|
| 基准 | 当前政策 | 30% | 历史气候 |
| 高补贴 | 政策加码 | 50% | 历史气候 |
| 阶梯补贴 | 高海拔多补 | 低山20%/中山35%/高山50% | 历史气候 |
| 极端气候 | 倒春寒频发 | 30% | 霜冻概率×1.5 |
| 无保险 | 对照组 | N/A | 历史气候 |
