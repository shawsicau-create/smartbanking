# Economist Analyst

## Purpose

运用经济学家思维框架分析经济现象、政策影响和市场动态。适用于经济学研究分析、政策评估和市场趋势分析。

## When to Use

- 需要经济学分析框架解读现象时使用
- 政策影响评估时触发
- 触发词："经济学分析"、"政策评估"、"市场动态"

## Instructions

### Step 1
确认分析类型：微观/宏观/政策/行业
### Step 2
选择经济学框架：供需分析/成本收益/博弈论
### Step 3
收集相关数据和经济指标
### Step 4
建模分析：因果推断、弹性计算、情景模拟
### Step 5
生成经济学分析报告

## Example Prompts

- "分析最低工资政策对就业的影响"
- "评估碳税对企业成本的传导效应"
- "分析房地产市场供需失衡的成因"

## Requirements

### Software
- Python 3.10+ or appropriate runtime

## Best Practices

1. **区分规范分析和实证分析**
2. **标注假设和模型的局限性**
3. **使用反事实推断而非简单相关**

## Common Pitfalls

- ❌ 混淆相关性和因果性
- ❌ 忽视模型的适用边界
- ❌ 过度简化复杂经济系统
