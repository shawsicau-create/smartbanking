### 2.2 Altman Z-Score（中国制造业私营企业适配版）

采用 Z'-Score 模型（适用于非上市制造企业）：

$$Z' = 0.717 \times X_1 + 0.847 \times X_2 + 3.107 \times X_3 + 0.420 \times X_4 + 0.998 \times X_5$$

| 变量 | 含义 | 计算 | 数值 |
|------|------|------|------|
| X₁ | 营运资本/总资产 | (1660-1040)/2800 | 0.221 |
| X₂ | 留存收益/总资产 | ≈960/2800 | 0.343 |
| X₃ | EBIT/总资产 | ≈383/2800 | 0.137 |
| X₄ | 股东权益账面值/总负债 | 1200/1600 | 0.750 |
| X₅ | 销售收入/总资产 | 4100/2800 | 1.464 |

> EBIT 估算：净利润250万 ÷ (1-25%税率) + 利息费用约50万 ≈ **383万**  
> 留存收益估算：所有者权益(1200万) × 80% ≈ **960万**

**计算过程**：
```
Z' = 0.717×0.221 + 0.847×0.343 + 3.107×0.137 + 0.420×0.750 + 0.998×1.464
   = 0.158 + 0.291 + 0.426 + 0.315 + 1.461
   = 2.651
```

**Z'-Score 判定标准**：

| 区间 | 判定 | 本企业 |
|------|------|--------|
| Z' > 2.90 | 安全区 | — |
| 1.23 < Z' < 2.90 | 灰色区（需关注） | ⚠️ **2.651（位于灰色区上端）** |
| Z' < 1.23 | 危险区 | — |

**结论**：Z'-Score = 2.651，处于灰色区上端，距安全区仅差 0.25 分，整体信用状况良好但需关注。

### 2.3 利息保障倍数（ICR）

```
年利息支出 = 500万 × 5.5% = 27.5万
ICR = EBIT / 利息支出 = 383万 / 27.5万 = 13.93 倍
```

| ICR 判定标准 | 本企业情况 |
|-------------|-----------|
| ICR > 5：利息覆盖充裕 | ✅ **13.93 倍，远超安全阈值** |
| 3 < ICR < 5：基本安全 | — |
| ICR < 3：偿债压力较大 | — |

若考虑企业现有负债利息（按存量负债800万 × 5%估算 = 40万），综合 ICR = 383/(27.5+40) = **5.67 倍**，仍属充裕水平。

偿还能力系数 = 经营性现金流 / 贷款本息 = 320万 / 527.5万 = 0.607

| 判定标准 | 分析 |

| 0.5 < 系数 < 1.0：部分覆盖 | ⚠️ **0.607，需关注** |

**分析**：年度经营性现金流（320万）仅覆盖贷款本息的 60.7%。但需注意：
